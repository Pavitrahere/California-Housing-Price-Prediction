# 🏠 California Housing Price Prediction

## 📌 Overview

This project analyzes housing data to predict property prices using machine learning techniques.

## 🎯 Objective

* Predict housing prices based on features like income, location, and population
* Identify key factors influencing house prices

## 🛠️ Tools & Technologies

* Python
* Pandas, NumPy
* Scikit-learn
* Matplotlib, Seaborn

## 📂 Dataset

* California Housing Dataset
* Contains features such as median income, house age, and location

## 🔍 Key Steps

1. Data preprocessing and cleaning
2. Exploratory Data Analysis (EDA)
3. Feature selection
4. Model training (Linear Regression / others)
5. Model evaluation

## 📈 Results & Insights

* Median income is a strong predictor of housing prices
* Model achieved good prediction performance

## 🚀 Future Improvements

* Try advanced models (Random Forest, XGBoost)
* Hyperparameter tuning
